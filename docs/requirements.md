# 功能需求文档 — 历史剪贴板

## 概述
Windows 桌面软件，自动记录用户复制的所有内容（文字和图片），提供历史查看、搜索、管理和再次粘贴功能。

## 功能列表

### F1 — 剪贴板自动监听
- 软件运行期间持续监听系统剪贴板
- 检测间隔：500ms
- 支持类型：纯文本（text/plain）、图片（image/png 等）
- 去重机制：内容与上一条记录完全相同时跳过写入
- 图片存储：以 PNG 文件保存到 `assets/images/`，文件名使用时间戳

### F2 — 历史记录展示
- 主窗口以卡片列表形式展示所有历史记录
- 排序规则：置顶卡片在最上方（按置顶时间降序），其余按复制时间降序
- 文本卡片：显示前 200 字符预览，超出显示 "..."
- 图片卡片：显示缩略图（宽 100%，高自适应，最大高度 150px）
- 每张卡片显示：复制时间（格式 "HH:mm:ss"）、内容预览、类型图标

### F3 — 搜索功能
- 顶部搜索框，输入关键词实时过滤
- 搜索范围：文本内容的全文模糊匹配
- 图片不参与搜索（但保留在列表中，搜索为空时显示）
- 搜索结果仍遵守排序规则

### F4 — 置顶功能
- 每张卡片有置顶/取消置顶按钮（📌 图标）
- 置顶卡片高亮显示（边框颜色不同，背景稍深）
- 置顶排序：最新置顶的排在最上面
- 置顶不受过期清理影响

### F5 — 删除功能
- 每张卡片有删除按钮（🗑 图标）
- 删除文字记录：从数据库移除
- 删除图片记录：从数据库移除 + 删除本地图片文件
- 无确认弹窗（操作可逆性低，接受误删；或可后续加撤销）

### F6 — 存储期限设置
- 顶部提供期限选择器：1 天 / 3 天 / 5 天
- 默认为 3 天
- 超过期限的记录在应用启动时自动清理
- 置顶记录永不过期
- 图片文件随数据库记录一并清理

### F7 — 再次粘贴
- 点击任意卡片，将其内容写回系统剪贴板
- 文字：写入 text/plain 格式
- 图片：读取本地文件，写入 image/png 格式
- 粘贴后窗口自动最小化到托盘（方便用户直接 Ctrl+V）

### F8 — 系统托盘
- 关闭窗口 → 隐藏到系统托盘（不退出）
- 托盘图标右键菜单：打开主界面 / 退出应用
- 双击托盘图标 → 打开主界面
- 托盘提示文字："历史剪贴板 - 运行中"

### F9 — 全局快捷键
- `Ctrl+Shift+V`：显示/隐藏主窗口
- 快捷键在应用启动时注册

### F10 — 开机自启
- 应用安装后默认开机自动启动
- 使用 electron auto-launch 实现
- （后续可加设置开关）
